Welcome to the README for the Script's Vanilla Extended - The Gourmet for 7 Days to Die! 
This mod aims to enhance and enrich your gaming experience by introducing new food & drink items.
Whether you're a seasoned survivor or just starting out in the post-apocalyptic world, this mod offers a fresh and exciting twist to your gameplay.

----------------------------------
	   Installation
----------------------------------

1. Backup: Before installing any mod, it's recommended to back up your game files and saves.
2. Download: Download the mod files from the provided source or website.
3. Locate Game Folder: Navigate to your 7 Days to Die game folder. This is typically located in C:\Program Files (x86)\Steam\steamapps\common\7 Days To Die.
4. Mod Folder: Inside the game folder, locate the "Mods" folder. If it doesn't exist, create it.
5. Install: Extract the mod files into the "Mods" folder. Make sure that the mod's folder structure matches the structure inside the "Mods" folder.
6. Launch: Start the game. The mod should now be loaded.

----------------------------------
	   Compatibility
----------------------------------

This mod is designed to be compatible with the latest version of 7 Days to Die as of its release. 
However, compatibility with other mods may vary. 
It's recommended to use this mod with a fresh installation of the game and to consult the mod author for any compatibility notes.
