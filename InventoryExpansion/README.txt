Version 1.0 compatible.

How to Install:

Copy folder "InventoryExpansion" in to your "%appdata%/7DaysToDie/Mods" path location.


NOTES:

There are couple changes in this mod.

1. Slots are increased to 72 from 45
2. Pack Mule perk has been modified to corresponding slots.
3. Pack Mule required Strenght levels has been adjusted to be more accurate. Levels are 1, 3, 5, 7, 10
3. Crafting que slots are increased to 8 from 4.
